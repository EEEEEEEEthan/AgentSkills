# 导出属性须有详细 hover 说明

## 原则

凡使用 `@export`、`@export_group`、`@export_subgroup` 等暴露到检视器的成员，**紧邻其上**须写 `##` 文档注释，内容足以让策划/同事在不读代码时理解：用途、单位、典型取值、与玩法或资源的约束。

无文档或仅一行泛化标题视为不合格。

## 写法

- 用 `##` 多行说明；首句可短，后续补单位、默认意图、与其他导出项的关系。
- 分组名可用 `@export_group("显示名", "前置说明")` 的第二参数给整组 hover；组内每个导出仍要单独 `##`。

```gdscript
## 平面内每秒移动速度（像素/秒）。用于追踪玩家前的巡逻位移。
@export var patrol_speed: float = 80.0

@export_group("武器", "射击相关参数；弹体场景须含 Bullet 根脚本。")

## 射击间隔（秒）。受全局攻速倍率乘算后再用于计时。
@export var fire_cooldown: float = 0.35
```

## 为什么重要

检视器悬停是策划调参与多人协作的主要入口；缺少说明会导致误用、重复沟通与错误工单。
